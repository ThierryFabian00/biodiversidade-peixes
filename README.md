# Biodiversidade de peixes

Pipeline de dados para coleta, tratamento e análise de registros de ocorrência de espécies de peixes disponibilizados pelo GBIF.

## Objetivo

Analisar a distribuição geográfica e temporal de espécies de peixes, começando por registros brasileiros de *Oreochromis niloticus*.

## Tecnologias

- Python
- Pandas
- API do GBIF
